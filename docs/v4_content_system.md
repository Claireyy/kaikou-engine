# Kaikou Cognitive Content System v4.0

## Definition

Kaikou v4.0 is a cognitive content production system focused on:

- automatic topic filtering
- automatic structure judgment
- automatic script generation
- viral probability scoring

Each run outputs:

1. Spreadable topics.
2. One-minute spoken scripts.
3. Viral probability scores.

## Input Sources

The system collects real platform signals from:

- Douyin: high likes, high comments, comment conflict.
- Xiaohongshu: high saves, dense emotion.
- Bilibili: knowledge videos, long comment chains.
- Zhihu: high-vote questions, structured discussion.
- Video accounts: real-life high-interaction content.

## Topic Filters

Only collect three types:

### Group Confusion

Many people are asking the same thing, but no one explains it clearly.

### Behavior Contradiction

People are doing something, but the logic is reversed.

### Reality Failure

Old experience no longer works in the new environment.

## Cognitive Models

Each topic can use exactly one main model.

Naval layer:

- Specific knowledge mismatch.
- Judgment.
- Identity structure.
- Leverage thinking.

Psychology layer:

- Cognitive dissonance.
- Attention residue.
- Habit path dependence.
- Behavioral automation.

Behavioral economics:

- Loss aversion.
- Framing effect.
- Anchoring effect.

## Reality Translation

Every model must be translated into daily scenes:

- delivery
- work
- messages
- relationships
- AI
- consumption
- decisions

No script should stay at the theory layer.

## Scoring

Each topic is scored 0 to 10 on:

- real-world frequency
- cognitive contrast
- emotional resonance
- comment controversy
- speakability

Total:

- 40-50: viral-level
- 30-39: shootable
- below 30: discard

## Spoken Script Structure

Each script follows:

1. Opening: real phenomenon only.
2. Scene: concrete daily image.
3. Cognitive explanation: one model only.
4. Reality translation: one sentence explaining the real reason.
5. Soft ending: no summary and no moral lesson.

## Language Rules

Must:

- sound spoken
- observe reality
- use one insight only
- include pauses
- use short lines

Must not:

- sound motivational
- sound like a psychology class
- stack concepts
- use AI-flavored explanation
- use long comma chains
