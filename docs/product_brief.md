# Product Brief

## Product Name

Kaikou Engine

## Purpose

Kaikou Engine turns internet behavior into a repeatable cognitive short-video production pipeline.

Each run produces four complete content packages for one-minute videos.

## Core Input

Internet attention signals from:

- Douyin
- Xiaohongshu
- Bilibili
- Zhihu
- Video accounts

## Core Output

For each run:

1. Four hot cognitive topics.
2. Four one-minute spoken scripts.
3. Four editing scripts.
4. Four subtitle-ready outputs.
5. Four publishing caption packages.

## Tone Rules

The output should feel like observing human behavior, not teaching a lesson.

Avoid:

- motivational tone
- academic explanation
- multiple models in one script
- more than one core idea per video
