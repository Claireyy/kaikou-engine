# Kaikou Cognitive Content Engine v6.1

## Core Upgrade

v6.1 expands the content system from human behavior only to six cognitive domains.

The goal is to reduce repeated topics and explain structural reality, not only personal emotion or behavior.

## Six Domains

### Human Behavior System

- judgment
- decisions
- habits
- cognitive bias
- behavior mismatch

### Social Structure System

- income structure changes
- class mobility changes
- city cost structure
- education return changes
- work-form changes

### Technology System

- AI replacement paths
- tools changing ability structure
- information access changes
- side effects of lower technical barriers

### Information System

- attention allocation
- information overload
- recommendation algorithms
- content homogenization
- cognitive bubbles

### Time System

- delayed feedback
- long-cycle returns
- compounding effect
- instant-gratification illusion
- time mismatch cognition

### Reality System

- rule changes
- environment reconstruction
- uncertainty structure
- systemic risk
- old experience failure

## Topic Structure

Each topic should follow:

```text
System change + real phenomenon + cognitive explanation
```

## Generation Rule

The script must explain structural causes.

It must not only explain:

- how people think
- how people feel
- how people behave

It must include:

- system change
- structural reason
- reality-level translation
- observable phenomenon

## Scoring

v6.1 uses a 60-point score:

- real-world frequency
- cognitive contrast
- emotional resonance
- comment controversy
- speakability
- system-change strength

Thresholds:

- 48-60: viral-level
- 36-47: shootable
- below 36: discard
